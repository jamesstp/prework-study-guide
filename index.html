Prework Study Guide

HTML

  • Gives the content structure
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements inlcuded in the head are not visible when a user looks at a webpage.

CSS

  • Gives the content style.
  • Language that is used to describe the presentation of website.

GIT

  • Free open space version control.
  • Allows web developers to collaborate on projects.
  • Won't overwrite each other's work.
  • Keeps track of your projects.

JavaScript

  • Lightweight programming language that is used to provide logic and interactivity.
  • Allows the user to interact with the webpage.
